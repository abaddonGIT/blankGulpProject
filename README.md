Starting a project for Gulp + Browserify
================

### Installation ###
    npm install
### Fast install ###
    yarn install
### Install venders ###
    gulp bower
### Dev work ###
    npm start
### Build the project ###
    npm run build
### Start eslint
    npm run lint    
### Load to server ###
    gulp sftp
### Generate sprites ###
    gulp sprites
### Generate svg sprite ###
    gulp svg
